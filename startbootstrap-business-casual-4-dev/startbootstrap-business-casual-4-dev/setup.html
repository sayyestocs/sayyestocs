Setup

Follow the steps below in order to complete setup

1. Download the text editor Atom. This is where you will write all your code.

2. Make a new file and call it whatever you want but before you save add the ending '.html'

3. For your styling page make another file and call it whatever you like preferably the same name and 'Style.css'
